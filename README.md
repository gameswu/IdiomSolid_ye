# IdiomSolid_ye 插件

> AstrBot 插件
>
> 这是一个提供给小夜的成语接龙小游戏插件

# 使用

## 指令组
所有指令均在指令组 `/成语接龙` 下

| 指令 | 参数 | 描述 |
|------|------|------|
| `/成语接龙 举行` | - | 开启一局成语接龙会话 |
| `/成语接龙 参加` | `<对局id>` | 参加成语接龙 |
| `/成语接龙 开始` | - | 开始一局成语接龙 |
| `/成语接龙 退出` | - | 退出当前的成语接龙 |
| `/成语接龙 结束` | - | 结束当前的成语接龙 |
| `/成语接龙 历史` | `<对局id>` | 查看指定对局的历史记录 |
| `/成语接龙 删除` | `<对局id>` | 删除指定对局（管理员权限） |    

## 配置项
| 配置项 | 描述 |
|--------|------|
| `maxPlayers` | 每局最大玩家数量 |
| `timeLimit` | 每轮时间限制（秒） |
